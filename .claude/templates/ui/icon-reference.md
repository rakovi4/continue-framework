# Icon Reference — Common Icons

Use Lucide icon library. In HTML mockups: `<i data-lucide="icon-name"></i>`. In React: `import { IconName } from 'lucide-react'`.

| Purpose | Lucide name | Usage |
|---------|-------------|-------|
| Help / question | `help-circle` | Help buttons, info tooltips |
| Checkmark | `check` | Feature lists, success indicators |
| Success circle | `check-circle-2` | Success states |
| Error circle | `x-circle` | Error states |
| Warning | `alert-triangle` | Warning banners |
| Alert info | `alert-circle` | Info/alert messages |
| Close | `x` | Close buttons, modals |
| Arrow right | `arrow-right` | CTA buttons, navigation |
| Arrow left | `arrow-left` | Back navigation |
| External link | `external-link` | External URLs |
| Key | `key` | Authentication, tokens |
| Power | `power` | Disconnect, logout |
| Refresh | `refresh-cw` | Retry, reload |
| Eye / toggle | `eye` / `eye-off` | Password visibility |
| Dashboard | `layout-grid` | Dashboard nav |
| Tasks | `clipboard-list` | Tasks nav |
| Activity | `clock` | Activity log nav |
| Board | `kanban` | Board nav |
| Profile | `user` | Profile nav |
| Settings | `settings` | Settings nav |
| Logout | `log-out` | Logout button |
| Loading | `loader-2` | Spinner (add CSS animation) |
